# ToyTopia - A Local Kids Toy Store Platform

**ToyTopia** is an online platform designed to help families discover and explore toys from local sellers. The website provides an engaging and playful shopping experience for kids and their parents while supporting local businesses. Users can browse toys, view detailed information, manage their profile, and interact with the platform in a secure and intuitive way.

## Purpose of the Project

The main goal of ToyTopia is to create a local online marketplace for kids' toys where families can:  

- Explore a variety of toys available from local sellers  
- Get detailed information about each toy, including images, ratings, and availability  
- Create and manage a user profile  
- Leave feedback or reviews for toys they are interested in  
- Experience a modern, responsive, and user-friendly interface  

This project is designed both as a learning exercise in modern web development and as a practical platform to showcase local toy collections.

---

## Key Features

- **User Authentication**: Users can sign up or log in using email/password or Google authentication powered by Firebase.  
- **Private Routes**: Certain pages like Profile or Add Toy are accessible only to logged-in users.  
- **Profile Management**: Users can update their display name and profile photo.  
- **Responsive Design**: Built with Tailwind CSS and DaisyUI to ensure a smooth experience on all devices.  
- **Interactive UI Elements**: Includes sliders, carousels, and scrolling marquees for an engaging interface.  
- **Notifications**: Feedback and alerts are displayed using React Toastify.  
- **Dynamic Toy Listings**: Browse toys with images, ratings, and available quantities.  
- **Smooth Animations**: Uses Swiper for interactive sliders and Marquee for scrolling content.

## Technologies and NPM Packages Used

- **React** – Frontend framework for building interactive UI  
- **React Router** – Handles page routing  
- **Tailwind CSS** – Utility-first CSS framework for styling  
- **DaisyUI** – Component library for building UI elements  
- **React Icons** – Icon library for React components  
- **React Toastify** – Notifications and alerts  
- **Swiper** – Carousel and slider components  
- **Firebase Authentication** – User authentication backend  
- **React Fast Marquee** – Scrolling text for announcements or promotions
